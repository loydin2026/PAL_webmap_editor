# 仙剑奇侠传一代 地图编辑器（网页版）

基于原 Win32 SDK 版本重写的纯前端网页版地图编辑器，无需安装，浏览器直接运行。

## 快速开始

### 1. 启动本地服务器

由于浏览器安全策略限制，本地资源文件需要通过 HTTP 协议加载。

#### 方式一：使用内置 Node.js 服务器（推荐）

在 `web-map-editor` 目录中，双击运行：

```
start-server.bat
```

或者命令行：

```bash
cd web-map-editor
node server.js
```

然后打开浏览器访问：**http://localhost:8080/**

#### 方式二：使用 VS Code Live Server 插件

在 VS Code 中安装 "Live Server" 插件，右键 `index.html` → "Open with Live Server"。

#### 方式三：使用 Python（如已安装）

```bash
cd web-map-editor
python -m http.server 8080
```

### 2. 自动加载资源

页面打开后会**自动扫描并加载**资源：

1. **Pat.mkf**（调色板文件）—— 自动加载
2. **所有 GOP 图组文件**（gop0000 ~ gop0226）—— 扫描为列表，按需加载缓存
3. **所有 MAP 地图文件**（map0000 ~ map0226）—— 扫描为列表，按需加载缓存

页面加载完成后，默认会自动打开第一个可用的地图和对应的图组。

### 3. 下拉列表操作

- **左侧「图组」下拉列表**：选择不同的 GOP 图组文件，加载对应的图块到网格中
- **右侧「地图」下拉列表**：选择不同的地图文件，加载对应的地图数据

当选中地图时，会自动加载其对应的 GOP 图组（编号匹配）。

### 4. 基本操作

| 操作 | 说明 |
|------|------|
| **左键点击** | 选择 Tile / 应用当前工具（画笔/擦除/障碍/模板） |
| **左键拖拽** | 选择工具模式下拖拽平移视图；画笔模式下连续绘制 |
| **右键点击** | 选中 Tile 并显示属性 |
| **Shift + 点击** | 将 Tile 添加到模板 |
| **滚轮** | 缩放（暂未完全实现） |

### 5. 工具说明

- **🔍 选择**：拖拽平移地图，Shift+点击创建模板
- **✏️ 画笔**：在左侧面板选择图块后，点击地图绘制
- **📋 模板**：使用 Shift+点击创建的模板，点击地图粘贴
- **🧹 擦除**：清除当前图层的 Tile 数据
- **🚧 障碍**：切换 Tile 的障碍标记
- **L0 / L1**：切换编辑的图层（第 0 层 / 第 1 层）

### 6. 文件保存

点击 "💾 保存" 按钮，会将地图数据下载为二进制文件（与原格式兼容）。

## 项目结构

```
web-map-editor/
├── index.html              # 主页面
├── css/
│   └── style.css           # 样式
├── js/
│   ├── palette.js          # 调色板解码（MKF 格式）
│   ├── gop-loader.js       # GOP 图组解码（RLE 解压）
│   ├── map.js              # 地图数据结构、坐标转换
│   ├── renderer.js         # Canvas 渲染引擎
│   ├── editor.js           # 编辑逻辑、状态管理
│   ├── ui.js               # 界面交互、下拉列表、缓存管理
│   └── main.js             # 入口、自动加载、主循环
├── assets/                 # 标记图像资源
│   ├── bitmap1.bmp         # 鼠标悬停标记（绿色）
│   ├── bitmap2.bmp         # 选中标记（蓝色）
│   ├── bitmap3.bmp         # 障碍标记（红色）
│   └── Object.bmp          # 人物对象预览
├── server.js               # Node.js 简易 HTTP 服务器
└── start-server.bat        # Windows 启动脚本
```

## 技术说明

- **解码器**：完整还原了原 C++ 代码中的 MKF 索引解析、RLE 解压、256 色调色板映射
- **坐标系统**：完全复刻了原 Win32 版本的等角（isometric）坐标转换算法
- **文件格式**：保存的地图文件与原编辑器 100% 二进制兼容（128×128×2×2 字节）
- **渲染方式**：使用 HTML5 Canvas 2D，双层等角绘制，支持透明色（Color Key）
- **缓存策略**：GOP 图组和地图数据按需加载并缓存，避免重复网络请求

## 注意事项

1. 浏览器必须支持 ES6（所有现代浏览器都支持）
2. 由于使用 `fetch` 加载本地资源，**必须通过 HTTP 服务器访问**，不能直接用 `file://` 打开
3. 地图文件和 GOP 图组文件必须配套使用（编号匹配，如 `map0001` 对应 `gop0001`）
4. 资源文件（Pat.mkf / gop/ / map/）需要在正确的相对路径上（`../../Pat.mkf`、`../../gop/`、`../../map/`）

## 原项目

本项目基于 palresearch 的仙剑奇侠传研究项目中的 Win32 SDK 版地图编辑器改写。

## 开发修改记录

- 修复 `Uint16Array` 溢出问题：GOP 文件偏移值可达 103980，改用 `Int32Array`
- 添加自动资源扫描：启动时自动通过 HEAD 请求扫描所有 GOP/MAP 文件
- 添加下拉列表：GOP 选择器和 MAP 选择器，支持实时切换
- 添加缓存系统：已加载的 GOP 和 MAP 数据缓存，避免重复网络请求
- 添加默认加载：自动加载第一个可用的地图和对应图组
